Current Focus: LibMessageKit

Since 2023, I've been working as an independent software developer and consultant, focusing on an innovative, end-to-end encrypted and anonymous messaging application. I've developed and maintain open-source projects including LibMessageKit, LibMessageKit-NetworkLayer, and SML (SecureMessageLayer).

Project Architecture

My current project is divided into three main components:

  • LibMessageKit: An open-source C library for building secure, end-to-end encrypted messaging applications.
  • LibMessageKit-NetworkLayer: A TCP Socket Networking Module.
  • SML (Secure Message Layer): A cryptographic module implementing DoubleRatchet and MLS protocols.

I've implemented the core library in C with WebAssembly integration for cross-platform compatibility. On the server side, I use virtualization with LXC, manage firewalls with NFTables, and utilize NGINX for server management. The system employs LibTLS (OpenBSD) for TLS socket communications and Libsodium for cryptography, supporting PQECDH.
